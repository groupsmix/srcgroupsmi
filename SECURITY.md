# Security Policy

## Supported Versions

Currently, only the main deployed version of GroupsMix is supported.

| Version | Supported          |
| ------- | ------------------ |
| Current | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

We take the security of GroupsMix seriously. If you believe you have found a security vulnerability, please report it to us.

Please email your findings to **security@groupsmix.com**.

### What to include
- A description of the vulnerability and its impact
- Steps to reproduce the issue
- Any relevant screenshots or logs

We will aim to acknowledge your report within 48 hours and provide updates as we investigate and resolve the issue. Please do not disclose the vulnerability publicly until we have had a chance to address it.
